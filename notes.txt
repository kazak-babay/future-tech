Все дробные значение округлять вверх

При выгрузке изображения с фигмы можно поставить коэффициент большой, чтобы картинка на большом экране смотрелась не убого, например 4х

css:

font-size меньше 14 -- преступление

рекомендуемый размер кнопки на телефоне 44Х44. Реализован через before в globals

button {white-space: nowrap} -- чтобы текст в кнопке не переносился

писать margin-bottom/right через селектор :not(:last-child) для возможных будующих перестановок

border-radius: 50%; -- круглая картинка

background: url() 'позиционирование' no-repeat var(color); -- элемент из url будет на фоне color

отсупы между текстом и элементами, которые расположенны вертикально, задаёт не тупо маржином у верхних, а с помощью display: grid + row-gap у родительского элемента
